# 📊 Sales Data Analysis Project

## 📌 About the Project
I created this project to understand how sales data can be analyzed using Python. The goal was to explore trends, identify top-performing products, and visualize the data in a simple and interactive way.

## 🚀 What this Project Does
- Generates a sample sales dataset  
- Cleans and prepares the data  
- Analyzes sales trends over time  
- Identifies top products and regions  
- Displays results using graphs and a dashboard  

## 🛠️ Tools Used
- Python  
- Pandas  
- Matplotlib  
- Streamlit  

## 📂 Project Structure
sales_data_analysis/   
│   └── sales_data.csv  
│  
├── notebooks/

│

└── README.md  

## ▶️ How to Run

### 1. Run the Notebook
- Open `sales_analysis.ipynb`  
- Run all cells to generate and analyze data  

### 2. Run the Dashboard
streamlit run dashboard.py  

## 📊 Key Insights
- Sales vary across months, showing possible seasonal trends  
- Products like Laptop and Monitor contribute more to revenue  
- Some regions perform better than others  
- Higher quantity sold generally increases revenue
